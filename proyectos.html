Mis proyectos

CompuLab

CompuLab+ está desarrollado con el framework Bootstrap para el diseño responsivo y moderno de la interfaz. Se emplean además librerías de DOM scripting para la interacción dinámica, junto con estructuras modulares en JavaScript que facilitan la gestión de datos y eventos.

MortyTK

Una Single Page Application con JavaScript Vanilla, integrando una REST API, manejo de rutas con hash routing, consumo de datos asincrónicos y estructura basada en componentes y renderizado dinámico.

Antidoto Edit

Antídoto Edit es la página de una editorial peruana comprometida con la cultura nacional. Este sitio web, desarrollado con HTML, CSS y JavaScript, emplea API's JSON para contenido dinámico. Descubre la pasión por la escritura en este antidoto.

Halloween

Landing page temática con diseño web responsivo, usando HTML5, CSS3, Flexbox y Grid, aplicando estructura semántica, animaciones suaves y buenas prácticas de maquetación moderna.
